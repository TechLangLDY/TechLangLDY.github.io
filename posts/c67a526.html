Markdown是什么？——一份简单易懂的教程

Markdown是什么？——一份简单易懂的教程

首先，请看一张动图：

有没有觉得很神奇？你只是在纯打字，按下换行的瞬间，没做任何操作，文本就自动拥有了美观的格式，左侧的【大纲】栏还自动生成了文章目录，整个过程美观又省心。

这就是Markdown，如果让我用简单的话来介绍它，我会这样告诉你：

Markdown 是一种写作方式，用这种方式写作时，只要在文本前添加特定的符号，就可以让文本自动变成特定的格式，从而为写作者省时省力。

1. Markdown介绍

1.1 适用场景

如果你要做的，是类似写毕业论文这种，对格式有很多特定要求的写作，那么很遗憾，Markdown并不适合这种写作任务，因为它的字体字号等样式都是已经设定好的，没办法自己再调，这是因为这种写作方式的初心就是让你不必自己调格式。

如果你要做的，是类似写报告发给他人、在个人博客里写文章、在电脑上做学习笔记这种，不对具体字号字间距做要求，只要文章整体美观有条理就可以的写作，那Markdown会是让你省时省力的好选择。

1.2 上手试一试

想学Markdown其实很快，如果你想使用Markdown，只需要：

  1. 找一款支持这种写作方式的软件
  2. 记住那几个写在文本前的特定符号

对于这两点，第二点是核心，只要懂得Markdown怎么用，用哪个工具都随便你。你可以在记住了这几个符号后，去体验各种不同的工具，比如我在文章开头给你做演示用的Typora。而这些符号其实自己敲几次很快就能记住，所以你可以边看这篇文章边跟我做，在读的同时顺手就把Markdown学了。

1.2.1 试手工具

我们可以用文章开头演示的Typora来试手，因为它的使用体验比较好。这个软件现在是收费的，但是之前的免费版本网上也有，并且免费版功能就已经够用，所以无论你是想用付费版还是免费版都可以满足需求。

最新版Typora有15天试用期，我们就选它来试手。首先进入Typora 官网 ，下拉到下载部分，下载并安装对应系统的Typora：

当然，如果你嫌下软件麻烦，也可以选个点击即用的线上工具，比如这个：Markdown在线编辑器,所见即所得 | 沃夫工具 (offso.com)。线上工具的差别只在它是左边打字右边实时预览，在学Markdown的效果上没什么区别：

1.2.2 试手样式

那么正式开始吧！你可以把你的试手工具和这篇知乎文章一左一右同时打开在电脑桌面上，然后跟着我一起敲这些符号。

我们按写一篇文章的逻辑顺序来：

a. 列全文粗纲-用标题样式

请在文本前输入：**# + 空格**。一个井号代表一级标题，两个井号二级标题，以此类推，一共可以到六级:

# 一级标题

## 二级标题

‘### 三级标题

…

###### 六级标题

输出效果：

一级标题

二级标题

三级标题

…

六级标题
b. 写正文细纲-用列表样式

我们可以通过分点陈列的方式来写一段正文的细纲，也就是使用“列表”格式，请在文本前输入：**+ + 空格**：

+ 第一个观点

输出效果：

  • 第一个观点

接下来，当你想另起一点时，只要按回车键换行就行了，下一行会自动生成一个小点点，当你想结束分点陈列时，就按两次回车键，退出列表格式。

+ 第一个观点

+ 第二个观点

+ 第三个观点

输出效果：

  • 第一个观点
  • 第二个观点
  • 第三个观点

如果你的细纲还分多个层级，那就打多级列表，也就是在回车健换行之后再按个Tab键，那这一行列表就会自动缩进一个空格，这时，如果你想回到上一级列表，就按Shift+Tab健。退出列表则依旧是按多次回车。

输出效果：

  • 第一个观点

    • 分论点一
    • 分论点二

  • 第二个观点

如果你想这个列表是有序的，那就将“+”号替换成“1.”，即在文本前输入：1. + 空格，后续操作与上方的无序列表一样：

  1. 第一个观点

输出效果：

  1. 第一个观点
  2. 第二个观点
  3. 第三个观点
d. 突出重点字词-用加粗样式

写正文时，我们会突出重点，可以把它加粗，只需把重点内容包在两个星号（**）中：

输入：*想要加粗的内容*

显示效果：想要加粗的内容

b. 突出重点段落-用引用格式

如果你想突出的是一个段落，那可以打一个像我上面这样，左边有一条线的文本格式，也就是“引用”格式。这需要在文本前输入：**> + 空格**:

> 自己要写的文本

输出效果：

自己要写的文本

e. 划分文章结构-用分割线样式

写完一部分后，我们可以加条分割线，使文章内容在视觉上结构更分明。这时，只需在单独的一行空行开头打上：—

输入：-–

输出效果：

好了，试敲就到这里，相信通过这几个例子你应该能明白Markdown文本怎么写了，平时写作时一般这几个格式也就够用了，如果你还想学更多的样式，比如表格、添加图片、添加超链接等，请前往本文《1.4 Markdown 的所有语法》部分。

1.3 Markdown 的所有语法

上文中，当需要不同格式时，我们需要往文本前添加不同符号，这种什么格式配什么符号的规则，我们叫它“语法”。Markdown 所有语法加起来也没多少，其中平时频繁用到的更少，因此非常好记，你可以在需要时去官网查对应的语法：Markdown 基本语法 | Markdown 官方教程](https://markdown.com.cn/basic-syntax/)。

事实上，Markdown还有更多玩法，比如做思维导图、甘特图，甚至是做PPT，这些等你在熟悉它以后，如果还感兴趣，可以再去探索。

2. Markdown 工具推荐

如果你选择的试手工具是线上的，你可能会有疑问：为什么我这个和文章开头的动图不一样？为什么我这个不是即打即显？这是因为即打即显并不是Markdownd工具的普遍功能，所以如果你想要这样的效果，就要选择相对应的工具。

Markdown工具分两类，一类是软件形式，比如 Typora，一类是平台形式，比如知乎简书之类的平台现在都支持使用Markdown。因此，这一部分主要推荐些好用的Markdown工具。

2.1 Typora

这款工具就是我在文章开头动图演示用的工具，它的一个最大特点就是即打即显，会自动生成文本大纲，并可以把文本导出成多种格式。它的官网是：Typora 官方中文站 (typoraio.cn)。这款好用的工具现在已经收费，89元三台设备买断，但是可以试用15天，你可以先试用看看。另外，如上文所说，typora早期的beta版还是免费的，其功能我个人是觉得就已经够用，你也可以去网上搜索相应资源。
